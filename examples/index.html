jQuery Simple Share

Basic Usage

Easy. Shares the current page, and you don't have to pass any options.

Call

$('.basic a').simpleShare();

Shape and Color Options

To change the shape or color of your icons, use the shape and color options.

$('.basic-2 a').simpleShare({'shape':'circle'});
$('.basic-3 a').simpleShare({'shape':'none', 'color':false});

Manually setting the share URL

simple-share automatically shares the URL of the current page, but if you want to set it yourself (say, to add referer=simpleshare or something like that) you can do it with the data-url attribute. To use the same URL for a set of simple-share links, add the URL you want to use in the data-url attribute of the DIRECT PARENT of the links.

Markup

<div class="manual" data-url="http://google.com">

Different URLs for different services

You can also specify different share URLs for each service. In each link, along with the data-service attribute, also include the url you want to share for that particular service. If you don't specify one, it will fall back to use the parent container's data-url, and if that doesn't exist, it will just use the current page.

<a data-service="facebook" data-url="http://yahoo.com">
